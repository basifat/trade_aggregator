# Trade Aggregator

Aggregates trades using the correlation id and decides to accept, reject or make the trade pending.

1. Run application:
   1. Cd into trade_aggregator
   2. pip3 install -r requirements.txt
   3. python run-app.py input.xml
  
2. Run Tests:
   1. Cd into trade_aggregator
   2. python -m pytest
